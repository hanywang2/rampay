---
title: "Create User"
description: "Set up a user profile with their information"
api: "POST /v1/partner/user"
---

### Body

<Param body="firstName" type="string">

First name of the user.

Example: `john`

</Param>

<Param body="lastName" type="string">

Last name of the user.

Example: `doe`

</Param>

<Param body="email" type="string">

Email address of the user.

Example: `john`

</Param>

<Param body="dateOfBirth" type="string">

The birth day of the user in format `yyyy-mm-dd`

Example: `1990-12-22`

</Param>

<Param body="isdCode" type="string">

The [ISD Code](https://www.indyatour.com/directory/isd-code) (country calling code) of the user

Example: `+1`

</Param>

<Param body="phoneNumber" type="string">

Phone number of the user

Example: `5417543010`

</Param>

<Param body="addressInfo" type="object">

<Expandable title="Toggle object">

<Param body="addressLine1" type="string">

Address Line 1

Example: `Rue de Forte`

</Param>

<Param body="addressLine2" type="string">

Address Line 2

Example: `Louvre Road`

</Param>

<Param body="state" type="string">

The state of the user's address

Example: `California`

</Param>

<Param body="city" type="string">

The city of the address

Example: `Los Angeles`

</Param>

<Param body="country" type="string">

The country (as a country code)

Example: `US`

</Param>

<Param body="postalCode" type="string">

The user's postal code

Example: `12345`

</Param>

</Expandable>

</Param>

<RequestExample>

```bash Request Example
curl --location --request POST 'https://api-sandbox.rampay.io/v1/partner/user' \
--data-raw '{
    "firstName": "john",
    "lastName": "doe",
    "email": "john_doe@example.com",
    "dateOfBirth": "1990-12-22",
    "isdCode": "+1",
    "phoneNumber": "5417543010",
    "addressInfo": {
        "addressLine1": "Rue de Forte",
        "addressLine2": "Louvre Road",
        "state": "California",
        "city": "Los Angeles",
        "country": "US",
        "postalCode": "12345"
    }
}
'
```

</RequestExample>
