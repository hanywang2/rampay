---
title: "Pricing"
description: "Create price"
api: "POST /v1/pricing"
auth: "none"
---

### Body

<Param body="fiatAmount" type="integer">

Amount of fiat currency.

Example: `200`

</Param>

<Param body="cryptoCurrencyCode" type="string">

Crypto currency as part of List of crypto currencies response.

Example: `USDT`

</Param>

<Param body="cryptoCurrencyNetwork" type="string">

The abbreviated code of the blockchain network.

Example: `eth` (for ethereum)

</Param>

<Param body="directory" type="string">

Whether to buy or sell

Example: `BUY_CRYPTO`

</Param>

<Param body="cfpmId" type="integer">

The fiat currency and the payment method that user wishes to pay with.

Example: `1`

</Param>

<RequestExample>

```bash Request Example
curl --location --request POST 'https://api-sandbox.rampay.io/v1/pricing' \
  --data-raw '{
      "fiatAmount": 200,
      "cryptoCurrencyCode": "USDT",
      "cryptoCurrencyNetwork": "eth",
      "direction": "BUY_CRYPTO",
      "cfpmId": 1
  }'
```

</RequestExample>
