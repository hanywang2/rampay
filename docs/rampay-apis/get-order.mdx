---
title: "Get Order"
description: "Fetch an order object using an identifier"
api: "GET /v1/partner/order"
---

### Auth Header

Please send you API Secret as Bearer token in Authorization header.

Header Key: `Authorization`

Header Value: `Bearer YourApiSecret`

### Parameters

<Param query="orderId" type="string" required>

The identifier of the order.

Example: `f4a685afd2d90ea8a27d380ae353aa8e`

</Param>

| ORDER STATUS |	Definition |
| ------------ | ----------- |
| `PENDING` |	Order gets created in this default state. |
| `PAYMENT_PENDING` |	User is attempting to pay for this order. |
| `PAYMENT_SUCCESS` |	Payment has been done by the user. |
| `RELEASING_CRYPTO` | Payment has been received for this order, and now we are releasing the crypto for the end user. |
| `COMPLETED` |	Payment has been completed and user has successfully received crypto in their wallet. |

<RequestExample>

```bash Request Example
curl --location \
  --request GET 'https://api-sandbox.rampay.io/v1/partner/order?orderId=b8c260068f1b2576fd3fdc4262b8bd80'
```

</RequestExample>
